JQuery.TextBox

Introduction

Combining a text input with a drop down select box

Project

  • Google Code: http://code.google.com/p/jquerytextbox/

  • Author: Dale Harvey (harveyd@gmail.com)

  • Tested Browsers: Firefox 2/3, IE 7/8, Epiphony 2, Opera 9, Safari 3

  • Notes : Uses absolute positioning, if something causes the input element to move $(window).resize(); will trigger the repositioning

Example


- click here to add a new Item

    $("input").textbox(
    {
        items    : ["item1","item2"],
        onChange : function(item)
        {
            alert("Changed : " + item);
        },
        onSelect : function(item)
        {
            alert("Selected : " + item);
        }
    });
    
    $("input").textbox("add","appended item");
